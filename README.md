# Pipeline de Vendas — SQLite + Python + Excel

## Problema
Automatizar a análise de vendas trimestrais e gerar 
relatório Excel sem intervenção manual.

## O que foi feito
- Banco de dados SQLite criado e populado via Python
- Consultas SQL para receita por mês, categoria e trimestre
- Relatório Excel gerado automaticamente com Python e openpyxl

## Tecnologias
Python | Pandas | SQLite | Matplotlib | OpenPyXL | Looker Studio

## Resultados
- Receita consolidada de 3 trimestres analisada
- Queda de 32% no Q3 2024 vs Q3 2023 identificada
- Dashboard interativo criado no Looker Studio
